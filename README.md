# EasyPack

欢迎使用EasyPack快速构建工程！

![](http://ww1.sinaimg.cn/large/63885f75ly1frh6f0fpmfg20g80mqguf.gif)

#### Start

```
$ npm install
$ npm install -g electron-packager
```



#### Dev

```
$ cd easy-pack
$ npm run start
```



#### Build

```
$ cd easy-pack
$ npm run build
```



EasyPack只专注在构建方面

- 效率：避免重复配置的工作
- 交互便捷：实现可视化操作
- 人性化：直接暴露配置，方便修改配置
- 专注开发
